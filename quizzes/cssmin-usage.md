# CSSmin quiz

A quiz about CSSmin.

---

CSSmin can be used to

|                                                           | `true` | `false` |
| ----------------------------------------------------------| ------ | ------- |
| make your CSS files smaller                               |  [x]   |  [ ]    |
| unminify your CSS files                                   |  [ ]   |  [X]    |
| minimize specific files or all css files                  |  [x]   |  [ ]    |
